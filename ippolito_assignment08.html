Introduction¶

This data set comes from the UCI Machine Learning Repository and includes evaluations of 115 Swiss engineering students. Data was collected while the students took six exams each. Data was collected about each student's keystrokes, mouse movements, and test times, along with how the student performed on the exam. I chose the data set because it included a good mix of variables, both numerical and categorical, with which to demonstrate various plot types.

Visualizations - Part 3 (Differences)¶

For starters, it took fewer statements to create the same plots in Seaborn, especially when it came to subplots. In Matplotlib, each subplot had to be initialized with a plt.subplot statement, while in Seaborn the axis could just be passed to the plot statement as an argument.

Seaborn also spaced the plots out better, while Matplotlib jammed the plots up against each other, even overrunning the x-axis labels with the plot below it. That required me to manually set the plot spacing, while Seaborn seemed to do it more intelligently on its own.

The thing that Seaborn didn't do very well automatically was decide binwidth. In one of the plots, the binwidth was so small that at first it looked like an empty plot. Adjusting it manually fixed the problem. In a different plot, there was an outlier with a very large negative number; Matplotlib handled this with no problem, while Seaborn churned on it for a full five minutes before I had to kill the kernel and figure out what it was doing. Removing the outlier fixed the issue, but it was interesting that Matplotlib handled it while Seaborn was kind of like Tom Cruise in A Few Good Men: It couldn't handle the truth.

Conclusions¶

Surprisingly, there seemed to be a positive correlation between the mean total grade and several variables: the amount of mouse movement in pixels, the number of keystrokes, the number of mouse clicks (both right and left), and the amount of time spent scrolling the mouse wheel. While it is doubtful that any of these factors directly influenced the student's grade, it is nonetheless interesting to note that students who performed better also demonstrated a greater amount of mouse activity.
